# Hikarino Deploy Bundle

This folder is the minimal static site bundle for deployment.

## Files

- `index.html`
- `privacy.html`
- `photo/`

## Deploy

Upload the contents of this folder as-is to any static hosting service.

Examples:

- Netlify
- Vercel static output
- GitHub Pages
- S3 static website hosting

## Notes

- `index.html` is the landing page.
- `privacy.html` is linked from the form and footer.
- Only the images referenced by the site are included in `photo/`.
