# 関連施設導線セクション デザイン設計書

## 実装アプローチ

`FACILITY` スライダー直下に、併設施設「みっかび東」へ遷移するための軽い紹介セクションを配置する。現行サイトのトンマナに合わせ、淡いブルーグレー面、中央寄せのコンパクトな濃いブルーの横長カード、白文字、白CTA、カード内右側の角丸写真を使い、FACILITYスライダーの補足導線として見える清潔感のある医療サイト表現にする。

### レイアウト構成

- **PC幅 (1024px以上)**:
  - `FACILITY` スライダーの直後に淡いブルーグレーの背景帯を敷く。
  - 中央に白い丸角カードを配置し、左に施設導線用の写真、右に紹介テキストとCTAを置く。
- **モバイル幅 (1024px未満)**:
  - カード内を写真、テキストの順に縦積みする。
  - CTAは横幅いっぱいに広げ、タップしやすい導線にする。

### 使用アセット

- 導線写真: `/photo/access_entrance.jpg`

### スタイリング (CSS)

- `related-facility-section` クラスで管理する。
- 背景色は `var(--bg-blue-pale)` を使い、現行LPの淡いブルーグレー面と揃える。
- 白いカード、控えめなシャドウ、`var(--primary-blue)` のラベルとCTAで医療サイトらしい清潔感を優先する。
- 既存トンマナから外れる装飾イラスト、過度な写真コラージュ、ベージュ・グリーン寄りの演出は使わない。

### 影響範囲

- `clinic/app/page.js` の `facility` セクション直後。
- `clinic/app/globals.css` の `RELATED FACILITY` スタイル。
- 既存の `Service` / `Facility` / `News` の配置は維持し、`Facility` と下部コンテンツの間に併設施設リンク導線を追加する。
